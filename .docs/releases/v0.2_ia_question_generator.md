# Release v0.2 - Gerador de Questões com Inteligência Artificial

## Objetivo
Expandir o projeto para utilizar Inteligência Artificial (Gemini) na geração automática de questões preparatórias para a prova Microsoft DP-700.

## Mudanças
- Adição das dependências `google-generativeai` e `python-dotenv` via `uv`.
- Configuração do cofre de chaves via arquivo `.env`.
- Reformulação da interface da página "Praticar Questões", organizando-a em abas:
  - 🎮 Responder
  - 📝 Cadastro Manual
  - 🤖 Gerar com Gemini
- Implementação da função `gerar_questao_com_ia()` usando o modelo `gemini-1.5-flash` para construir cenários práticos em formato JSON.

## Status
- **Finalizado**: A geração de questões usando a API do Gemini está totalmente integrada à base de dados de estudos local (arquivos CSV).
