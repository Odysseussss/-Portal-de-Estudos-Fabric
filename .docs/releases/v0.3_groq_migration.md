# Release v0.3 - Migração para Llama 3 (via Groq)

## Objetivo
Resolver os problemas de limites de quota e modelos restritos da API gratuita do Gemini migrando o gerador de questões para a API do Groq (que é incrivelmente rápida e robusta).

## Mudanças
- Substituição da biblioteca `google-generativeai` pela `groq` no ambiente fechado (gerenciado pelo `uv`).
- Modificação do setup da API: do `.env` antigo onde usávamos `GEMINI_API_KEY`, agora o sistema espera um `GROQ_API_KEY`.
- Atualização da função subjacente de geração de questões para invocar o modelo `llama3-70b-8192` do Groq via ChatCompletions. Adotou-se o parâmetro `json_object` garantindo respostas nativas no formato estrito de JSON sem risco de markdown extra na string.
- Ajuste das interfaces, alertas e descrições na interface do Streamlit para referenciar a solução mais nova.

## Status
- **Finalizado**: O gerador de questões por IA está ativo com a nova integração e muito mais veloz.
